The old almnjoy.dev was a WordPress site cosplaying as Windows 98. Cute for a week, a maintenance tax forever. So it's gone, and the real one lives here now, right next to the projects and the daily log.

Same idea, less baggage. Long-form writeups land here weekly, or whenever I actually finish something worth showing. The short stuff, the jots and links and half-formed questions, live on the [daily page](/daily). This is where the ideas get room to breathe.

## Why kill it

A themed WordPress install is a hobby that pretends to be a website. Plugins to patch, a theme to babysit, a database to back up, and a login I forgot half the time. The novelty of the Win98 chrome wore off long before the upkeep did.

What I actually wanted was a place to write, versioned like everything else I build, that deploys itself when I push. No dashboard. No update-nag. Just markdown.

## What this is instead

This blog is a content collection in the same Astro repo as the rest of allemanit.com. A post is a markdown file. I commit it, Cloudflare Pages builds it, it's live. The whole thing is static, fast, and boring in the best way.

The plan from here:

- **Weekly** posts, or on-demand when I ship something cool.
- **Daily** notes, ideas, and goals over on `/daily`, which runs more like a living profile than a feed.
- One voice across both. No em dashes, no filler, honest about what's still rough.

That's it. The blog is back, minus the costume.
