This is the part I am most actively building: a set of AI agents wired into the way I actually work.

- **Agent roster.** Specialized agents, each owning a job, instead of one assistant trying to do everything.
- **Help desk.** An agent that handles support intake, triage, and first-line answers.
- **Infra agents.** Tiered agents that watch and act on infrastructure, with approval gates before anything changes state.
- **Agent brain.** A shared memory layer (Cognee) so the agents keep context, decisions, and history across sessions instead of starting cold every time.
